# CLAUDE.md

This is an MSPixelPulseAgency demo website repo for `mspixelpulse-demo-home-services`.

## Project

NorthStar Home Services is a fictional, mobile-first home services demo website for plumbers, electricians, HVAC, cleaning, painting, roofing, landscaping, handyman businesses, and other local service companies.

## Rules

- Work only inside this repo.
- Keep the site mobile-first, accessible, simple, and service-lead focused.
- Use placeholder details only: Toronto, Ontario, +1 (000) 000-0000, hello@mspixelpulse.com.
- Do not use real client data.
- Do not use copyrighted images.
- Run `npm run build` before committing.
- Push and deploy to Vercel only after build passes.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Vercel settings

- Framework: Vite
- Install: npm install
- Build: npm run build
- Output: dist
- Root Directory: ./

## Quality standard

The website should be clean, fast, accessible, and easy to use on mobile. It should prioritize service discovery, quote requests, phone calls, service areas, pricing clarity, reviews, FAQs, and simple UX.
